Question 1: greet()

Write a function greet that takes a String argument, name. The function should RETURN the string "Hello, [DYNAMIC-NAME], how are you?"

Argument Info
Name Type Description
name String The person's name

Example Code

When you enter this code:

console.log(greet('Tom'))
console.log(greet('Jerry'))

You should see this in the console:

Hello Tom, how are you?
Hello Jerry, how are you?

Question 2: getRating()

Write a function getRating that takes 2 arguments: a String argument, movieTitle, and a String argument, review. The function should LOG two lines. On the first line should be either stars or "???", depending on the review, and the second line should be a string "Movie: [DYNAMIC-MOVIE-TITLE]".

The function has no return value.

Here's how to decide what stars to show:
  • 'bad' or 'awful': ☆
  • 'good': ☆☆
  • 'great': ☆☆☆
  • Literally anything other string: '???'

Feel free to copy the emojis for the stars, or use an asterisk *

Argument Info
Name Type Description
movieTitle String The title of the movie
review String A single word opinion on the film

Example Code

Below are some functions and their output

getRating('Cats', 'bad')
☆
Movie: Cats
getRating('Cars', 'good')
☆☆
Movie: Cars
getRating('Gravity', 'odd')
???
Movie: Gravity

Question 3: hideAndSeekCounter()

Write a function hideAndSeekCounter that takes a Number argument, endNumber, which will be bigger than 0. The function should LOG several strings. First, it should count up to the given number starting from 1, with "..." after each number. Then, on the final number, it should add "!" to it instead. Finally, one last line of "Ready or not, here I come!"

See the code examples below if you're confused

Argument Info
Name Type Description
endNumber Number A number greater than 0

Example Code

hideAndSeekCounter(4)
1...
2...
3...
4!
Ready or not, here I come!

Question 4: cheer()

Write a function cheer that takes a String argument, word.

The function should LOG several strings. For every letter of word make a line print: "Give me a [UPPERCASED-LETTER]", where the letter has been uppercased. Once the entire word has been shouted, print the line "What's it spell?" and then one last line, where you repeat the word three times, each with an exclamation point and all uppercase.

See the code examples below if you're confused

Argument Info
Name Type Description
word String A lowercase word with no spaces or punctuation

Example Code

cheer('win')
Give me a W!
Give me a I!
Give me a N!
What's it spell?
WIN! WIN! WIN!

A Little Advice

For many of you, this is your first time dealing with a technical challenge. It will seem overwhelming, but believe it or not, you already know everything you need to solve these puzzles.

Read over the questions and then take a minute. Remember, before attempting to solve any of these with code, solve them with your words. Feel free to go back over the previous lessons (especially the final one on problem solving), this is open book!

Remember, this test is not about memorization, it's about problem solving. It's not about having the BEST answer, it's about finding any answer. If you get stuck, feel free to Google. However, we want to know what you know, not what the internet knows. If you copy code into your answers (which is allowed!) just make sure that you can explain what it does when your interviewer asks you about it.

The only wrong answer is the answer you can't exlpain. So, take another breath, think before your code, and good luck!
